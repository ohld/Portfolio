59 научная конференция МФТИ. Лучший доклад в секции.

    Секция компьютерной безопасности и защиты информации. Доклад "Использование технологии распознавания лиц для решения задач информационной безопасности."

    Был разработан прототип программы, который открывал доступ к секретным документам, используя свежий снимок с веб камеры. Были предложены идеи защиты от "атаки фотографией".

Instabot - обертка над API и скрипты для СММ в Instagram.

    Open-source проект, написанный целиком на Python, с помощью которого Вы можете не только раскрутить свой аккаунт в Instagram, но и собирать разного рода информацию из этой социальной сети. В репозитории есть достаточно много готовых скриптов для совершенно разных задач, так что, скорее всего, Вам даже не придется писать код. Более 100 звезд на Github (март 2016).

Хакатон по обработке изображений в МФТИ. Первое место.

    Участникам (17 команд) было предложено решить две задачи:

  • Написать алгоритм классификации, определяющий фильтр принятого на входе изображения произвольного размера. Иcпользовалось 11 фильтров Instagram.
  • Создать алгоритм, предлагающий «наиболее подходящий» нейрофильтр для исходного необработанного изображения. Предлагать можно любой из 10 заданных нейрофильтров из приложения PhotoLab.

    • Результат первого задания рассчитывался автоматически, вторую задачу команды защищали перед членами жюри.

Décodage d’une formule de pricing. Седьмое место.

    Участникам (900+ команд) было предложено подобрать формулу / алгоритм, по которому вычислялась стоимость страховки авто, используя исторические данные как о машинах, так и пользователях.

    Мое решение данной задачи было презентовано на конференции datafest3 в рамках ML тренировок.
